# Тетрис

### Как запустить игру?
```python main.py```

### Как настроить игру? 
В файле ```config.py``` лежат настройки игры. 
Можно без опасений менять следующие параметры: 

        +----------------------+--------------------------------------------------------+      
        |  background_color    |   цвет поля                                            |
        |  second_color        |   цвет оформления                                      |
        |  borders_color       |   цвет границ кубиков                                  |
        |  cube_colors         |   список из цветов кубиков                             |
        .-------------------------------------------------------------------------------+
        |  start_speed         |   начальное время обновления падения в мc              |
        |  max_speed           |   максимальное время обновления падения в мс           |
        |  speed_step          |   шаг времени обновления падения в мс                  |
        |----------------------|--------------------------------------------------------|
        |  screen_width        |   количество кубиков, размещаемых на доске в ширину    |
        |  screen_height       |   количество кубиков, размещаемых на доске в высоту    |
        |  size                |   размер кубика в пикселях                             |
        +----------------------+--------------------------------------------------------+
        |  borders             |   размер границ кубика в пикселях                      |
        |  gap                 |   отступы между кубиками в пикселях                    |
        +----------------------+--------------------------------------------------------+

        .---------------.----------------.-----------------.------------------------.--------------------.----------------------------.
        |  S  |  DOWN   |    A  | LEFT   |    D  | RIGHT   |   E    |   W   | UP    |         Q          |          ПРОБЕЛ            |
        |---------------|----------------|-----------------|------------------------|--------------------|----------------------------|
        | Опустить вниз | Сдвинуть влево | Сдвинуть вправо | Вращать против часовой | Вращать по часовой | Форсированно опустить вниз |
        *------------------------------------------------------------------------------------------------*----------------------------*
