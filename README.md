# Тетрис

### Как запустить игру?
```python main.py```

### Как настроить игру? 
В файле ```config.json``` лежат настройки игры. 

        "background_color": цвет поля 
        "second_color": цвет оформления 
        "cube_colors": список из цветов кубиков 

        "start_speed": начальное время обновления падения в милисекундах
        "max_speed": максимальное время обновления падения в мс 
        "speed_step": шаг времени обновления падения в мс

        "screen_width": количество кубиков, которых можно разместить на доске в ширину
        "screen_height": количество кубиков, которых можно разместить на доске в высоту
        "size": размер кубика в пикселях

        "margin": 200. Отступ с информацией. 


### Как играть? 

Вы можете совершать следующие действия с фигурой:


            S               A               D                 E                         Q
    
    Опустить вниз   Сдвинуть влево  Сдвинуть вправо   Вращать против часовой    Вращать по часовой
